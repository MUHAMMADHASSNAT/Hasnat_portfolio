# Public Folder

Yahan static assets rakho jo directly accessible honge.

## Profile Image

Apni `profile.jpg` file ko yahan copy karo:
- `public/profile.jpg`

Ya agar tumhare paas already kisi folder me hai, to:
1. `profile.jpg` file ko copy karo
2. `public` folder me paste karo
3. File ka naam exactly `profile.jpg` hona chahiye

## Projects Images (Optional)

Agar tum projects add karna chahte ho, to:
- `public/projects/` folder banao
- Project images ko wahan rakho
- Example: `public/projects/project1.jpg`
